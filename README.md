# JEDI

**Journalistic Excellence through Domain-specific Intelligence**

## Installation

```bash
pip install jedireporter
```

For development:

```bash
pip install -e ".[dev]"
```

## Usage

```bash
jedireporter --version
```

## Development

### Running Tests

```bash
pytest
```

### Running Linting

```bash
flake8
```

### Running All Checks with Tox

```bash
tox
```
