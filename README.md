# JEDI

**Journalistic Excellence through Domain-specific Intelligence**

## Installation

```bash
pip install jedi
```

For development:

```bash
pip install -e ".[dev]"
```

## Usage

```bash
jedi --version
```

## Development

### Running Tests

```bash
pytest
```

### Running Linting

```bash
flake8
```

### Running All Checks with Tox

```bash
tox
```
